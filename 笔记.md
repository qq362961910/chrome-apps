###权限

* activeTab
```
描述： 当用户跟该扩展交互时，扩展仅在活动标签被赋予额外权限。
“用户交互”包括:
   1. 用户点击扩展的浏览器按钮或者页面按钮
   2. 用户选择了扩展的上下文菜单
   3. 用户敲击了扩展定义的快捷键
```

* declarativeContent
```
描述： API 根据网页内容采取行动，而不需要读取页面内容的权限。
可用版本： 仅用于 Dev 分支。
权限： "declarativeContent"
```
